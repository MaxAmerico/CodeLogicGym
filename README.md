# 🧠 CodeLogicGym

Welcome to CodeLogicGym — a personal training ground for sharpening programming logic using Python.

This repository contains a collection of logic exercises organized into folders. Each exercise includes both the solution code and an about.md file explaining the idea behind the problem.

Think of it as a gym for your brain — where each problem is a workout for logical thinking. 💪

## 📂 Repository Structure

Each folder represents a programming logic exercise.

Inside each exercise folder you will find:

exercise-name/

mycode.py   # My exercice/code I did
      <hr>
solution.py # The solution of the code
      <hr>
about.md    #More info about the exercice
      <hr>

This structure makes it easy to:

Understand the problem context

Analyze the solution

Track progress through different exercises

## 🎯 Goals of This Repository

Improve programming logic

Practice problem solving

Build stronger Python fundamentals

Document learning progress

This repository works as a personal learning log and may also help others practicing programming logic.

## 🧩 Types of Exercises

The exercises may involve:

Conditionals

Loops

Mathematical logic

Basic algorithms

Pattern recognition

Problem decomposition

More exercises will be added over time as part of continuous practice.

## 📌 How to Use

You can explore the exercises in different ways:

Read the about.md file first to understand the problem.

Try solving it yourself.

Check the solution.py to compare approaches.
